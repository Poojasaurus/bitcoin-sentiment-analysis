# Bitcoin Sentiment vs Hyperliquid Trader Performance

## Objective
Analyze the relationship between Bitcoin Fear & Greed sentiment and trader performance from Hyperliquid historical trading data.

## Datasets
1. fear_greed_index.csv
2. historical_data.csv

## Methodology
- Data cleaning
- Date extraction and merging
- Exploratory Data Analysis (EDA)
- PnL analysis by sentiment
- Win-rate analysis
- Trade volume analysis
- Coin-level analysis
- Trader ranking

## Key Questions
- Does sentiment affect profitability?
- Are traders more profitable during Fear or Greed?
- Which coins perform best under different sentiment regimes?
- How does trading activity change with sentiment?

## Deliverables
- Jupyter Notebook
- Visualizations
- Business Insights
- GitHub Repository
